# Sherv OS Integration

MotionGuard participates in Sherv OS as a public product repository.

- ChatGPT: executive brain, strategy, prioritization, final review.
- Cursor: supervised implementation and visual debugging.
- Codex: tightly scoped background tasks.
- Linear: operational truth.
- GitHub: technical truth.

## Controls
1. Read the repository and linked Linear issue before editing.
2. Keep scope bounded and prefer existing architecture and dependencies.
3. Run native lint, typecheck, tests, build, and relevant browser checks.
4. Protect user input, telemetry, and any motion/accessibility data from unnecessary capture.
5. Production deploys, public publishing, credentials, billing, and destructive changes require human approval.
6. Releases document validation, remaining risk, rollback, and required demo assets.
7. Missing integrations fail closed and are reported explicitly.
